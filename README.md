# 🎬 cinema-prompt

A Claude Code skill that generates **one deeply-crafted AI video concept** in a cinematic, mythologized-subculture style — with ready-to-use prompts for **Kling video** and **Midjourney start-frames**.

Skill для Claude Code, который генерирует **одну глубоко проработанную концепцию AI-видеоролика** в кинематографическом стиле — с готовыми промптами для Kling и Midjourney.

---

## Что это

Каждая концепция — **6–7 красивых самодостаточных кадров одного мира**, связанных стилем и героем, но **не связанных нарративом**. Это не короткий фильм с сюжетом, а **альбом икон** из одного мифологизированного места. Эталоны жанра — мифологизированные субкультуры мира (парижская преступная аристократия, бразильская фавела-мифология, японская якудза + дзен, мексиканский картельный магический реализм, постцифровой реализм робота).

Скилл выдаёт:

- Концепцию в одном из трёх режиссёрских темпераментов: **«Икона»** / **«Действие»** / **«Сюр»**
- Раскадровку из 6–7 кадров по 7 ролям (establishing-тэблоу, group portrait, macro detail, hero portrait, surreal collision, symbolic still-life, silhouette/back-shot)
- Для каждого кадра: **Midjourney start-frame prompt** + **Kling video prompt** + камера/длительность
- **Style anchor** и **character lock** (копируются дословно во все кадры — даёт сквозную фактуру)
- Workflow генерации с `--sref` / `--cref` для cross-frame consistency
- Sound design: музыка, diegetic, SFX

## Установка

Скопируй папку `cinema-prompt/` (с файлом `SKILL.md`) в директорию скиллов Claude Code:

**Windows:**
```
%USERPROFILE%\.claude\skills\cinema-prompt\
```

**macOS / Linux:**
```
~/.claude/skills/cinema-prompt/
```

После перезапуска Claude Code скилл будет доступен как `/cinema-prompt`.

## Использование

```
/cinema-prompt <тема>
```

**Примеры тем:**

- `/cinema-prompt гонконгские триады в эпоху TikTok`
- `/cinema-prompt человекоподобный робот в Бразилии`
- `/cinema-prompt робот-андроид в Москве наводит суету`
- `/cinema-prompt сибирские дальнобойщики-староверы`

Без аргумента скилл предложит 5 неожиданных тем-провокаций.

## Жанровая ДНК

Скилл калиброван по 6 эталонным раскадровкам:

| Эталон | Мир |
|---|---|
| **Bandit** | Парижская преступная аристократия. Маски, оружие, классические интерьеры. |
| **BanditTruck** | Mad Max-десертный налёт. Бандит в бинтах, золотая статуя в фуре. |
| **Brazil** | Фавела-мифология. Christ Redeemer с Узи, мачете крупно, ряд автоматов. |
| **Japan** | Якудза + дзен. Татуированные тела в онсэне, питон на плече босса, кои. |
| **Mexico** | Картельный магический реализм. Конь под водой, голубь на стволе АК. |
| **Robot** | Постцифровой реализм. Робот в магазине, робот за рулеткой, робот арестован. |

Общий код: **реальное место (страна/субкультура) + миф (статус, ритуал, насилие, духовность, анахронизм)**.

## Визуальный язык

- **Формат:** всегда **4:3**, плёночное зерно, halation на тёплых источниках, anamorphic flares
- **Палитра:** один тёплый акцент + глубокая тень (без серых нейтралок)
- **Свет:** только практический (лампы, свечи, фары, голые лампочки, неон, заходящее солнце)
- **Композиция:** симметричные тэблоу, низкий угол героя, сверхширокий для эстаблишинга, макро для тактильности
- **Бренд в кадре НЕ генерируется** — пользователь добавляет логотип/название в постпродакшне (After Effects)

## Структура раскадровки

### Правило 5/5
В каждой раскадровке должны быть представлены минимум 5 ролей кадра:
- [x] establishing-тэблоу
- [x] макро-детайл
- [x] сюрреальная коллизия
- [x] симметричная композиция
- [x] silhouette / back-shot

### Ритм-карта
Длительности кадров чередуются (например 3s/5s/10s/5s/7s/8s), не плоско. Кадр 1 — короткий резкий **hook**, не establishing. ≥4 из 6 кадров с активной камерой.

## 11 законов prompt-инжиниринга

Выведены из боевых тестов. Подробно — в `SKILL.md`.

1. **Якорь первым** — иконографический субъект в первое предложение, отдельно
   - 1a. **Макро: объект, а не действие** — главный субъект макро — объект, не герой с объектом
2. **Один сюр-якорь, без вложенных «X внутри Y»** — два объекта рядом, не вложенно
3. **Сквозной style anchor** — идентичная style-фраза, скопированная во все 6 промптов
4. **Character lock** — 3 маркера героя, копируются в каждый кадр
5. **Никакого бренда в кадре** — всё в постпродакшне
6. **Лимит 3 нарративных элемента на кадр**
   - 6a. **MJ не считает выше 3-4** — для групп использовать «approximately N»
7. **Светотеневые коллизии явно** — что чем освещено и откуда
8. **Первый кадр = HOOK, не establishing**
9. **Никакой статики** — motion в каждом кадре, ≥4/6 активная камера
10. **Животные-проводники в widely shot дропаются** — либо главный субъект, либо отдельный кадр
11. **Two-subject через положение, не через роли** — `frame-left:` / `frame-right:` вместо «X в driver seat, Y в passenger seat»

## Workflow генерации

1. Сначала генерируется **Hero portrait** в Midjourney — он определит фактуру и героя
2. URL получившегося изображения подставляется как `--sref <url>` во все остальные промпты концепции (даёт сквозную палитру/фактуру)
3. Туда же `--cref <url>` для character lock (где есть герой)
4. В Kling: использовать MJ start-frame как первый кадр через **Image-to-Video** — Kling наследует атмосферу

## Что нужно

- [Claude Code](https://claude.ai/code) (CLI или одна из IDE-интеграций)
- Доступ к **Midjourney v7** (для генерации start-frames)
- Доступ к **Kling 2.x Pro** (для video, режим Image-to-Video)
- After Effects или аналог — для финального брендинга поверх кадров

## Доработка под свой вкус

Скилл откалиброван под эстетику **tartariot** (мифологизированные субкультуры, ночной синий + амбер, плёночное зерно, 4:3). Чтобы адаптировать под другой стиль:

1. Замени блок **«Жанровая ДНК»** в `SKILL.md` — опиши свои эталоны
2. Перепиши **«Визуальный язык»** под свою палитру/формат/свет
3. Адаптируй темпераменты «Икона/Действие/Сюр» под свои нужды
4. Законы 1–11 — универсальны, их можно оставить как есть (они про модель, не про эстетику)
