# 🎬 cinema-prompt

Claude Code skill: **фото шмотки + вайб бренда → промо-констелляция уровня арт-дирекшена** — 6 кадров одного мира с готовыми промптами для **Midjourney** (start-frames) и **Kling** (video).

---

## Что это

Вход: **одно фото вещи** (куртка / штаны / аксессуар / лук) ± описание бренда (своя марка, референс-бренды или просто mood).
Выход: **констелляция из 6 кадров одного мира** — не сюжетный фильм, а альбом икон, где каждый кадр самодостаточен и цепляет отдельно.

Скилл сам проводит триангуляцию **шмотка × бренд → мир** (16 brand-архетипов × 10 семейств локаций) и собирает россыпь по WOW-доктрине.

## WOW-доктрина (ядро)

Целевая эмоция зрителя: **«ВАУ, я такого не видел — лайк»**. Формализована в 10 правил (W1–W10 в `SKILL.md`), главные:

1. **Пересказ ≤5 слов** — каждый кадр пересказывается фразой, вызывающей «ЧТО?!» («конь висит под вертолётом»). Фраза = заголовок кадра.
2. **Коллизия миров** — ВАУ несёт столкновение несовместимого (мотоцикл × бабушкина гостиная, робот × казино), а не локация/погода/вайб.
3. **Deadpan** — невозможное снято как скучная хроника (CCTV, гейм-движок, home video). Камера не удивляется — удивляется зритель.
4. **Соучастие** — минимум 1 кадр делает зрителя участником (POV, пасть/взгляд в линзу).
5. **Эскалация** — пик невозможного на кадрах 3–5, не в начале; финал — послевкусие.
6. **Одна невозможность на кадр** — обычность продаёт невозможность.
7. **Шмотка в двух режимах** — Mode A «вещь нормальная, мир сошёл с ума» / Mode B «мир будничный, вещь — невозможный гость».

## Структура россыпи

- **Кадр 1 — HOOK** (fisheye-конфронтация или POV-вовлечение) — фиксирован
- **Кадры 2–5 — из ролевого пула** под коллизию: COLLISION ICON (обязателен), POV IMPLICATION, TOTEM, DETAIL, CREW, WAIST, DARK ICON, WORLD DEADPAN
- **Кадр 6 — AFTERMATH** (послевкусие) — фиксирован

Жёсткого шаблона порядка больше нет — скелет меняется от россыпи к россыпи.

## Эстетика

**«Снято, не отрендерено»**: жёсткий флэш, просроченная плёнка, CCTV, bodycam, PS2-движок, radical noir. НЕ глянцевая каталожка, НЕ luxury-кампания, НЕ perfume ad. `--style raw --s 50`, аспект 4:3 (мастер, кропится в 9:16).

Калибровка — по 12 эталонным раскадровкам с миллионными просмотрами (папка References; ДНК-таблица в `SKILL.md`).

## Использование

```
/cinema-prompt <фото шмотки> [описание бренда/вайба]
```

Скилл триггерится сам на любое фото одежды с намерением сделать видео-контент.

## Workflow

1. Кадр 1 (HOOK) генерируется в MJ первым — задаёт фактуру.
2. Style Reference (drag-and-drop Кадра 1) + Character Reference (фото шмотки) вручную добавляются в остальные промпты в интерфейсе MJ. В тексте промптов `--cref/--sref/--cw` НЕ дублируются.
3. MJ-кадр → Kling Image-to-Video (5s/10s, Pro).
4. Бренд/лого/текст — только в постпродакшне (After Effects), в промптах запрещены.
