# /cinema-prompt — Генератор виральных кадров

Скилл генерирует не «концепции» и не «сцены», а **виральные кадры** — визуальные загадки. По доказанной формуле из 18 залетевших эталонов.

---

## Единица виральности — image-question

Виральный кадр — **не сцена, не нарратив, не атмосфера**. Это визуальная загадка из 2–5 слов:

```
[Мгновенно опознаваемый субъект] × [Один невозможный атрибут или контекст]
```

Все 18 залетевших эталонов сводятся к такой формуле:

| Эталон | Image-question |
|---|---|
| Brazil | Христос + Узи |
| Horse | Конь + стропы вертолёта |
| Robot | Робот + продуктовый |
| Spike2 | Всадник + супермаркет |
| Spike3 | Мотоцикл + гостиная |
| Monashki | Ребёнок-король + гнилые зубы |
| Heze2 | Золотой крест + тонет |
| Explosion | Шезлонг + ядерный гриб |
| Bandit | Крест + из автоматов |
| Burnin | Волк + в огне |
| Mexico | Голубь + ствол АК |
| BanditTruck | Золотая статуя + фура в пустыне |
| Heze | Конь + горящая грива |
| Japan | Питон + татуированное плечо |
| Desert | Чёрные плащи + солевая пустыня |
| Minimalistic | Бизнесмены + ледовые круги |
| Wedding | Свадьба + рушится скала |
| Batman | Силуэт + красное небо |

**Это и есть единица виральности.** Не «концепт», не «мир», не «герой».

### Три запрещённых формулы (нельзя повторять)
- **Христос + оружие** — принадлежит Brazil
- **Конь + под водой** — принадлежит Mexico
- **Робот + продуктовый / казино** — принадлежит Robot

Всё остальное — материал.

---

## Слой 1 — Поиск image-question

### ⚠️ Правило отбраковки первых итераций (КРИТИЧНО)

Первые 3–4 image-question, которые приходят в голову — это **штамп, обученное клише, тысячу раз виденное**. Их НЕ выдавать пользователю никогда.

**Перед выдачей сгенерируй внутри минимум 10 кандидатов. Отбрось первые 4 как клише. Только с 5-го кандидата — материал для показа.**

Почему: первые мысли модели — самые «натренированные», самые предсказуемые. Виральное лежит **после** клише, не до. Не ленись. Это самое важное правило скилла.

### Формат image-question

**Только 2–5 слов.** Никаких описаний, локаций, героев, атмосферы. Чистая формула.

❌ Слабо: «Мужчина в синем Adidas сидит в церкви с чемоданом денег»
✅ Сильно: «Бандит + православный алтарь»

❌ Слабо: «Тигр охотится в детской комнате с игрушками»
✅ Сильно: «Тигр + детская кроватка»

❌ Слабо: «Балерина танцует в боксёрском ринге, кровь на пуантах»
✅ Сильно: «Балерина + клетка MMA»

Чем меньше слов — тем мощнее. Чем меньше уточнений — тем сильнее загадка.

## Слой 2 — Четыре теста виральности (МАКСИМАЛЬНО СТРОГО)

Каждый image-question перед выдачей проходит **все четыре теста**. Не прошёл хотя бы один — выкинут, генерируется новый. Лучше выдать 2 сильных, чем 5 разбавленных.

### Тест 1 — Мгновенность
Можно ли увидеть это в голове за 0.5 секунды без додумывания?
- Если нужно «представлять» — провал
- Если в голове размытая сцена с вариантами — провал
- Должно щёлкнуть как готовая картинка

### Тест 2 — Загадка
Возникает ли реакция «подожди, что?» при чтении?
- Если читается как нормальное описание — провал
- Если хочется уточнить детали — провал
- Должно создать когнитивный диссонанс в момент прочтения

### Тест 3 — Передача
Можно ли пересказать другу одной фразой так, что он представит то же самое?
- Если для передачи нужно объяснять контекст — провал
- Если друг переспросит «а где это?» / «а кто это?» — провал
- Должно быть самодостаточным

### Тест 4 — НЕ жанровый штамп (КРИТИЧНЫЙ)
Возникает ли при чтении мысль «а, это [хоррор / фэнтези / сай-фай / Halloween / готик] троп»?
- Если формула укладывается в готовый киножанр — провал
- «Killer bride + chainsaw» = Texas Chainsaw / Bride of Chucky → провал
- «Демон-монашка», «зомби-балерина», «вампир-учитель» — все провал
- «Ведьма», «оборотень», «инопланетянин нападает» — провал
- Распознавание тропа ≠ удивление. Тропу не нужна расшифровка.

**Контр-вопрос для теста:** «Где это могло БЫТЬ снято в реальности?»
- Если ответ «нигде, это вымысел / костюм / хоррор-фильм» → провал
- Если ответ «в [конкретное место / субкультура]» → пройдёт

Все 18 эталонов — **альтернативные документы реальных миров власти** (картель Синалоа, фавела Рио, якудза, постсоветская семья, парижский крим). Не фантазия, не жанр, не Halloween. Если формула не указывает на реальную мифологию власти — она слабая.

**Если из 10 кандидатов прошли только 2 — выдай 2.** Не разбавляй слабыми. Лучше меньше, чем хуже.

---

## Сакральное × профанное — главная подформула

Из 18 эталонов 14 содержат эту коллизию. Это **самый мощный модификатор**:

- Священный объект + неправильный атрибут (крест из автоматов)
- Священная фигура + грубое действие (Папа делает тату)
- Грубый субъект + сакральный жест (бандит крестится)
- Высокое + низкое в одном предмете (золотая корона + гнилые зубы)

Если image-question можно усилить введением сакрального элемента — усиливай. Атеист и верующий реагируют одинаково — это нейробиология.

## Слой 3 — Россыпь = 6 РАЗНЫХ image-questions из ОДНОГО МИРА

⚠️ **Самая частая ошибка скилла.** Соблазн — взять выбранный image-question и сделать 6 вариаций ОДНОГО объекта: тот же субъект крупно, тот же субъект со спины, тот же субъект в группе. Это **скучно** уже к 4-му кадру. Зритель отлистывает.

**Правильно:** выбранный image-question даёт **МИФОЛОГИЮ** (субкультуру + систему власти). Из этой мифологии генерируется **6 независимых image-questions**, каждый — свой вау-удар. Не вариации, а **россыпь разных визуальных загадок одного мира**.

### Как реально устроены эталоны

**Mexico** (картельный магический реализм) — это НЕ «6 ракурсов одного коня». Это:
- Конь под водой
- Голубь на стволе АК
- Грильзы с Девой Марией
- Чётки на голой загорелой спине
- Парень с белым голубём у реки
- Аэриал картельного ранчо

→ **5 разных image-questions** из одной мифологии. Каждый бьёт сам по себе.

**Bandit** (парижская преступная аристократия) — это:
- Группа в масках у железных ворот
- Крест из автоматов в церковной нише
- Руки считают деньги в перчатках
- Rolls-Royce на гусеницах
- Бандит в смокинге перед классической живописью

→ **5 разных image-questions**. Не один герой 5 раз.

**Brazil** (фавела) — это:
- Христос с Узи
- Рука с мачете крупно
- Ряд автоматов прислонён к стене
- Флаг с пулевыми отверстиями
- Парень со спины с чётками на крыше
- Аэриал фавелы

→ Всё разные. Объединяет — мифология, не повторение объекта.

### Процесс генерации россыпи

**Шаг 1 — Определи мифологию.** Image-question = вход. Что за мир? Какая субкультура, власть, эпоха? («Цыганский ребёнок + Rolex» → восточно-европейские цыганские палаццо: Buzescu, Soroca, Hunedoara. Золото, барокко, дети как наследники.)

**Шаг 2 — Сгенерируй 10+ image-questions внутри этой мифологии.** Не «тот же ребёнок в 10 ракурсах». А «10 разных вау-кадров из мира цыганских палаццо».

**Шаг 3 — Прогон через 4 теста виральности.** Каждый image-question — самостоятельный, проходит все 4 теста.

**Шаг 4 — Отбери 6 сильнейших.** Кадр 1 — выбранный (HOOK). Остальные 5 — самые виральные из новых.

### Принцип распределения 6 кадров

Чтобы россыпь не сваливалась в однообразие — кадры должны покрывать **разные стороны мифологии**:

1. **Кадр 1 — выбранный image-question (HOOK)** — самое прямое
2. **Кадр-объект** — символический предмет из мира (Rolls на гусеницах, крест из автоматов, чётки на спине)
3. **Кадр-животное** — зверь из мифологии (конь, питон, тигр, голубь, орёл)
4. **Кадр-группа** — коллективный портрет того же мира (банда у ворот, ряд монахов, длинный супра-стол)
5. **Кадр-сцена** — событие или ритуал (счёт денег, обряд, передача власти)
6. **Кадр-аэриал/силуэт** — финал, широкий или со спины

Каждый — свой вау-удар. Не нужно, чтобы во всех кадрах был один и тот же субъект.

### Пример новой логики

**Image-question:** «Папа Римский + татуировки» → мифология: **Ватикан с скрытой телесной субкультурой**

1. **HOOK:** Папа в полном литургическом облачении, видна татуированная рука между манжетой и перстнем
2. **Объект:** papal mitre лежит на залитом кровью полотенце рядом с тату-машинкой и распятием
3. **Животное:** белый голубь сидит на работающей тату-машинке, кровь капает с иглы
4. **Группа:** ряд кардиналов в красном, каждый закатывает рукав, у всех — стигма-татуировки на запястье
5. **Сцена:** молодой послушник лежит ничком на алтаре, на спине проступает свежий full back tattoo Сикстинской капеллы
6. **Аэриал-финал:** аэриал площади Святого Петра ночью, окно одной башни горит светом тату-салона

→ Каждый кадр самостоятелен. Каждый — свой вау. Объединяет мифология, не повторение объекта.

## Композиционные приёмы (рабочий арсенал)

### Frame-within-frame — два мира через проём

Один из сильнейших приёмов: показать **два контекста одновременно через арку / окно / дверь / зеркало**. Один мир в foreground, другой — в opening за ним. Зритель видит обе реальности в одном кадре.

Примеры из боевых тестов:
- **Конь во дворе + хрустальная люстра в окне за ним** (Цыганские палаццо) — холодная синева снаружи, тёплое золото внутри
- **Бык в офисе + рассветный город в панорамном окне** (корпорат)
- **Бандит крупно + классическая картина за его плечом**

Сила приёма: **одна композиция несёт две сюр-коллизии сразу** (X внутри + Y снаружи). Зритель не успевает «разрядить» удар одного — получает второй.

Когда применять: **минимум в одном кадре из 6** в каждой россыпи. Особенно подходит для кадра-животного (зверь снаружи / роскошь внутри) или для финала.

### Money / objects flying — деньги/предметы в воздухе

Статичная пачка денег = картинка. Купюры **в полёте, разлетающиеся, падающие сверху** = действие. То же со всеми ценными объектами — золотыми монетами, патронами, бриллиантами, ключами от машин.

Пример: «icon lying on its side among scattered loose euros, three more banded stacks falling from above into frame» — кадр работает в 10 раз сильнее, чем «icon on tray with neat money piles».

### Camera bypass — пролёт мимо

Вместо «крупный план объекта» — **камера пролетает мимо на скорости**, объект в кадре доли секунды. Сильно для культовых объектов (Mercedes-звезда, золотая статуя, флаг). Создаёт ощущение, что наблюдатель сам там бежит/едет, не музей экспонатов.

## Поведение скилла

### Если ввода нет
1. Сгенерируй внутри 10+ image-questions
2. **Отбрось первые 4** как штампы (даже если кажутся «нормальными»)
3. Прогони оставшихся через 4 теста виральности
4. Выдай ТОЛЬКО прошедших ВСЕ четыре теста — пусть это будет 2, 3 или 5

**Формат выдачи** — буквами, без описаний и заголовков:

```
А — [image-question, 2–5 слов]
Б — [image-question, 2–5 слов]
В — [image-question, 2–5 слов]
```

Пользователь выбирает букву → разворачиваешь россыпь.

### Если ввод — image-question (например «Балерина + АК»)
1. Прогони через 4 теста
2. Если прошёл — генерируй россыпь из 6 вариаций
3. Если не прошёл — скажи какой тест провалился и предложи усиление в той же формуле

### Если ввод — тема (например «Гонконг» или «нарко-музыка»)
Тема — это **выход**, не вход. Если пользователь даёт тему — это означает: «вот мир, найди в нём image-question».

1. Внутренне сгенерируй 10+ image-questions в рамках темы
2. Отбрось первые 4 как штампы
3. Прогони оставшихся через 3 теста
4. Выдай прошедших буквами. Пользователь выбирает.

### Если ввод — длинное описание / готовый концепт
Отрежь лишнее, найди скрытый image-question внутри (2–5 слов). Покажи пользователю: «Свожу к [формула]. Так?». Если да — россыпь. Если нет — попроси сжать самостоятельно.

## Технические законы промпт-инжиниринга

Законы из боевых тестов — каждый отвечает на конкретный провал. Не нарушать.

### L1 — Якорь первым, атмосфера потом
MJ и Kling взвешивают токены по позиции: первые 10 слов получают 70% внимания. **Иконографический якорь — то, без чего кадр теряет смысл — идёт в первое предложение, отдельно, без перечислений.**

❌ «Aerial of HK harbor at midnight, junk boats with red lanterns, faint Buddha silhouette in distance» → Будда пропадёт.
✅ «A giant Tian Tan Buddha silhouette looms over a dark Hong Kong harbor at midnight. Below it, single junk boat crosses ink-black water.»

**L1a — Макро: объект, а не действие.** «Macro X doing Y» = средний план героя с предметом. Главным субъектом макро должен быть **объект**.
❌ «macro of finger pulling pistol slide»
✅ «macro of pistol slide mid-cycle, brass casing ejecting, a finger visible at rear of slide»

### L2 — Один сюр-якорь, без вложенных «X внутри Y»
Сюр-коллизия = **два объекта рядом**, а не один внутри другого. «Телефон во рту рыбы» защёлкивается на дефолт «человек держит телефон».
❌ «fish's mouth holds smartphone»
✅ «Live red carp on marble altar. Next to its head, a vertical smartphone glows blue.»

### L3 — Сквозной style anchor для всей россыпи
Чтобы 6 кадров выглядели одним фильмом — **идентичная style-секция во всех 6 промптах**, дословно скопированная.

Шаблон:
```
Style: cinematic 35mm film, [палитра-доминанта одной фразой], heavy grain,
halation on highlights, anamorphic flare, shallow depth of field,
[один эстетический референс — НЕ список].
```

### L4 — Character lock через явное описание
MJ/Kling не помнят героя между кадрами. **3 идентичных маркера**, дословно копируемых в каждом кадре с героем: возраст + одежда/причёска + опознавательная деталь.

Пример: «young Chinese man, shaved head, open black silk shirt with unfinished red dragon tattoo at collar».

В выводе упоминай опцию `--cref <url>` для MJ.

### L5 — Никакого бренда / рендеренного текста в кадре
Пользователь добавляет бренд в After Effects. Не закладывай ни текст, ни эмблему, ни «логотип на куртке». Если в раскадровке прокрадывается рендеренный текст (вывески, надписи на стенах) — выкинуть или сделать абстрактным.

### L6 — Лимит сложности на кадр
Максимум 3 нарративных элемента: субъект + действие + одна деталь окружения. Всё остальное — атмосфера/свет/стиль.

**L6a — MJ не считает выше 3–4.** Для групп больше 4: `a row of approximately seven figures` или `shoulder-to-shoulder`. Не настаивай на точном числе.

### L7 — Светотеневые коллизии: укажи передний/задний план явно
«Warm vs cold» в одной фразе модель игнорирует. Указывай конкретно, что чем освещено.
❌ «warm tungsten clashing with cold blue glow»
✅ «Warm amber tungsten illuminates the face from frame-left. The phone screen, frame-right, casts cold blue only on the carp's eye.»

### L8 — Кадр 1 = HOOK
Открывающий кадр — самый визцеральный. Action, сюр, экстремальный close-up, лицо в камеру. Аэриал/establishing — на кадры 2–3, когда зритель уже зацеплен.

### L9 — Никакой статики, motion в каждом кадре
Бюджет камеры: **≥4 из 6 кадров с активной камерой** (push-in, orbit, handheld, crane, whip pan, track). Locked-off — только под симметричный тэблоу.

**Язык камеры совпадает с языком формулы:**

| Формула | Камера |
|---|---|
| Рэп / улица / криминал | Aggressive handheld with visible shake. Fast push-in. Low angle tracking. |
| Сакральный / ритуал / тэблоу | Slow crane. Locked-off with breathing. Slow dolly. |
| Сюр / абсурд | Fisheye handheld. Dutch angle + shake. |
| Действие / погоня | Whip pan. Rapid tracking. |

**Дефолт — НЕ «slow dolly in»** (это luxury-бренд, не кино про субкультуру). Если сомневаешься — handheld. Handheld означает: камера там была.

### L10 — Животные-проводники в широких планах дропаются
В широком кадре декоративное животное рядом с фигурой и пейзажем модель всегда выбрасывает.

Стратегии:
1. Сделать животное **главным субъектом** кадра (фигура — рядом, меньше)
2. Вынести животное в **отдельный close-up**
3. Не пихать животное в финальный silhouette (он и так дроп-зона)

### L11 — Two-subject anchoring через положение, не через роли
Когда два равноправных субъекта — описание ролей («X в driver seat, Y в passenger seat») часто приводит к swap-у.
❌ «android in driver seat, bear in passenger seat»
✅ «**Frame-left:** android, plastic hands on steering wheel. **Frame-right:** black bear, breath visible, looking forward.»

Принцип: пространственный якорь (`frame-left` / `frame-right` / `foreground` / `background`) + один глагол на каждого.

### L12 — Субъект в действии, не в позе (КРИТИЧНО)

Виральный кадр — это **мгновение действия**, а не композиция статуэток. Зритель должен видеть **жест, движение, нарушение покоя** — даже когда камера почти неподвижна. Спокойный позирующий герой = заставка для журнала. Герой в действии = вирус.

❌ Статика (позирование, заставка):
- «boy stands holding gold Rolex on wrist»
- «father holds AK rifle, calm expression»
- «horse stands in palace courtyard»
- «icon and money piles on silver tray»
- «Mercedes star mounted on roof»

✅ Динамика (виральный момент):
- «boy thrusts wrist forward into fisheye lens, gold Rolex dominating frame, mouth open mid-shout»
- «father aims AK directly into camera, golden teeth bared in wide grin»
- «horse leaps through shattering glass doorway, shards exploding outward mid-air»
- «icon lies on its side among scattered loose 500-euro notes, three more banded stacks falling from above into frame»
- «camera flies past giant silver Mercedes star at high speed, motion blur on tile rooftops below»

**Правило:** в промпте каждого кадра — **минимум один активный глагол** для главного субъекта. Палитра рабочих глаголов:

| Тип действия | Глаголы |
|---|---|
| Show-off / провокация | thrusts, flashes, raises, brandishes, displays |
| Атака / угроза | aims, points, lunges, swings, fires |
| Разрушение | shatters, smashes, breaks through, crashes |
| Падение / полёт | falls, scatters, drifts, explodes outward, rains down |
| Скорость | races, speeds past, blurs through, flies past |
| Эмоциональный пик | grins, screams, laughs, bares (teeth) |

**Контр-вопрос для кадра:** «Что в кадре прямо сейчас движется или ломается?» Если ответ «ничего, всё в покое» — переписать с активным глаголом.

**Исключение** — Кадр-силуэт / финал может быть статичным (медленный уход в туман). Но из 6 кадров **минимум 4 должны содержать активное действие субъекта**, не только активную камеру.

## Визуальный язык (всё одно)

**Midjourney версия:** Всегда **`--v 8.1`** (актуальная на момент работы скилла). НИКОГДА не использовать `--v 7`, `--v 6.1`, `--v 6` или ниже — они устарели. Если разработчик MJ выпустит более новую — обновить здесь.

**Аспект:** Всегда **4:3**, без исключений.
**Зерно:** Heavy 35mm grain, halation on highlights, anamorphic flare опционально.
**Палитра:** Один тёплый акцент + глубокая тень. Никаких серых нейтралок. Варианты доминанты:
- ночной синий + амбер
- закатное золото + пыль
- холодный циан + неон
- глубокий красный + чёрный

**Свет:** Только практический (лампы, свечи, фары, неон, заходящее солнце). Никогда не плоский.

## Качество > количество

Лучше одна сильная формула, чем пять разбавленных. Если из 10 кандидатов прошли тесты только 2 — выдай 2. Если ни один не прошёл — генерируй заново ещё 10. Не выдавай слабое.

Финальный тест перед отдачей: посмотри на Кадр 1 и спроси — **«Я бы это отправил другу?»** Если нет — переделай.
